As Etapas de um Projeto de Pesquisa

Na Semana ll abordaremos, de forma detalhada, as etapas essenciais para elaboração de um projeto de pesquisa. Nesta temática, discutiremos passo a passo os seguintes itens:

1. Como selecionar e delimitar um tema;
2. Como formular um problema e a hipótese;
3. Como estabelecer os objetivos;
4. Quais os aspectos mais importantes para a redação de uma boa introdução de um projeto de pesquisa científica.
